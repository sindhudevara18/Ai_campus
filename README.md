# 🏆 AURA – AI Campus Companion  

AURA is an AI-powered campus companion designed to support students academically, professionally, and emotionally all in one platform.
It combines intelligent learning tools with campus engagement features to enhance productivity, reduce stress, and improve student success.

 ## Academic Copilot
- AI-generated topic explanations  
- Flashcard generation for active recall  
- Quiz generation for self-testing  
- Smart study planner creation  

## Mood & Stress Support
- Daily mood tracking  
- Mood trend visualization  
- Encouragement & support suggestions  

## Campus Info Module
- AI Notice Summarizer  
- Internship Alerts dashboard  
- Active Clubs & community updates  

## Tech Stack
- React.js  
- TypeScript  
- Node.js / Express  
- Tailwind CSS  
- Framer Motion  
- Recharts  
- LLM API Integration  

## Problem Statement
Students often struggle with unstructured study methods, exam stress, and staying updated with campus opportunities.  
AURA integrates AI-powered academic tools with campus engagement features in a single platform.

## How to Run Locally
(in terminal)
npm install
npm run dev
